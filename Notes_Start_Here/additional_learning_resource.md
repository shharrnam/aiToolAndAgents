## The only learning sources you need provided by Calude & OpenAI themselves ( ++ The will to google / or ask GPT your doubts to get started )

### If you prefer Claude

https://docs.claude.com/en/api/overview - General learning and overview.

https://docs.claude.com/en/docs/get-started  - Technical details around the claude api & how it can be configured to multiple things, image / files  / tools  etc

https://github.com/anthropics/courses/tree/master/anthropic_api_fundamentals  - Fundamental practical examples, from making your first api, call to building a simple chatbot, almost every example is available here, everything that any paid course provides is exactly this, but this is from Claude themselves, so yeah better.

https://github.com/anthropics/claude-cookbooks  - More detailed practical examples again almost anything you want to learn, is already available here such as building database agents, RAG search, Voice agents  / Image agents / MultiModal Agents from claude and partners like ElevenLabs, Pinecone + a lot more!

### If you prefer OpenAI

https://platform.openai.com/docs/overview - Overview about everything, textual learning with examples

https://platform.openai.com/docs/api-reference/introduction - Technical learning / references for APIs, image models, fine tuning etc.

https://github.com/openai/openai-cookbook/tree/main - Practical learning Cookbook by OpenAI, again anything you can think of is already available here, from basic chatbots, to voice agents, RAG search, database agents, how to finetune models

https://cookbook.openai.com/  - Openai cookbook user friendly with a UI, the github one is better as they are practice codebases which you can run and create step by step.
